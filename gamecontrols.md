### Form Element

In settings window, when a form element / input element is selected, you can use widely known keys like _<kbd>Delete</kbd>_, _<kbd>Backspace</kbd>_, _<kbd>‚Üê</kbd>_, _<kbd>‚Üí</kbd>_. The following keys are also supported:

| ‚å®Ô∏è Key                                | Function          | Element Type |
| ------------------------------------- | ----------------- | ------------ |
| <kbd>Enter</kbd> / _<kbd>Space</kbd>_ | Toggle checkbox   | Checkbox     |
| <kbd>‚Üë</kbd> / <kbd>‚Üì</kbd>           | Increase/decrease | Number input |
| _<kbd>‚Üí</kbd> / <kbd>‚Üë</kbd>_         | _Increase_        | Range input  |
| _<kbd>‚Üê</kbd> / <kbd>‚Üì</kbd>_         | _Decrease_        | Range input  |
